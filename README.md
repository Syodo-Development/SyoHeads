# SyoHeads

<img width="85" height="85" alt="grafik" src="https://github.com/user-attachments/assets/ed21cc52-b838-4aa8-9ae4-cd95c7cc1c83" />

SyoHeads is a plugin that adds playerheads to PowerNukkitX.
Get heads from bedrock players using /head bedrock (player)
Or get heads from java players using /head java (player)

You also can get heads from a heads database by using /head database
